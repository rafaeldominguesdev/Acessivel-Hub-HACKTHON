# AcessívelBR Hub 🚀
**Projeto Desenvolvido para o Grand Prix SENAI de Inovação - Desafio Petrobras**

O **AcessívelBR Hub** é uma plataforma inteligente que centraliza e orquestra iniciativas de acessibilidade dispersas em grandes organizações, transformando processos manuais em uma estratégia ágil e baseada em dados.

---

## 🎯 O Desafio
A Petrobras identificou que suas iniciativas de acessibilidade (RH, Saúde, Ergonomia, TI) ocorrem de forma **descentralizada e isolada (silos)**. Isso resulta em processos lentos, falta de integração e baixa visibilidade de problemas sistêmicos.

## 💡 Nossa Solução: O Hub de Centralização Inteligente
Nossa plataforma atua como o **conector central**, unindo as áreas e utilizando IA para:
*   **Centralização Real:** Unifica dados de RH, TI e Saúde em um comando único.
*   **Triagem e Roteamento via IA:** Direciona demandas para o responsável certo em segundos.
*   **Clustering (Padrões):** Identifica falhas que se repetem em diferentes unidades para tratá-las de uma vez só.
*   **IA de Apoio:** Sugere soluções baseadas em dados para o gestor decidir com rapidez.
*   **Mapa de Calor:** Visibilidade geográfica total dos gargalos de acessibilidade.

---

## 🛠️ Tecnologias
*   Next.js 16 + React 19 + TypeScript
*   Tailwind CSS + Framer Motion
*   IA para Análise e Clustering

## 🚀 Como Executar
O código principal está na pasta `/grandprix`.
1.  Aceda à pasta: `cd grandprix`
2.  Instale: `npm install --legacy-peer-deps`
3.  Rode: `npm run dev` (Porta 7777)
